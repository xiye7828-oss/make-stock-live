# 一、误区拆解型

> 结构公式：很多人以为……→ 其实更该看……→ 为什么会错 → 错了会导致什么 → 正确理解是什么
> 适合老师风格：理性拆解型 / 强观点提醒型

## 案例 1

- **适合主题**：股价下跌就是买入机会？
- **适合段落**：第 3 段（拆第一个大误区）
- **讲解顺序**：
  1. 很多人以为：跌了就是便宜，便宜就该买
  2. 但更该看的是：跌的原因是什么
  3. 为什么会错：因为不知道是调整还是趋势反转
  4. 错了会导致什么：抄底抄在半山腰，越套越深
  5. 正确理解是什么：先看位置，再看逻辑，不急着接飞刀
- **核心讲点**：「跌了 ≠ 便宜，便宜 ≠ 可以买」这个等式要拆开

## 案例 2

- **适合主题**：MACD 金叉死叉真的有用吗？
- **适合段落**：第 3 段（误区拆解）
- **讲解顺序**：
  1. 很多人以为：MACD 金叉买、死叉卖，稳稳盈利
  2. 但更该看的是：MACD 是滞后指标，不是预测指标
  3. 为什么会错：因为它是跟随价格出现的，已经走了一段了
  4. 错了会导致什么：总是慢半拍，买在最高点附近
  5. 正确理解是什么：它更适合用来确认趋势，不适合用来预判
- **核心讲点**：技术指标是跟随者，不是领导者

# 二、框架教学型

> 结构公式：今天讲一个判断框架 → 为什么需要这个框架 → 框架的2-4个点 → 每点解释 → 一句总结
> 适合老师风格：老师带教型 / 理性拆解型

## 案例 1

- **适合主题**：如何判断一只股票的位置高低
- **适合段落**：第 4 段（上第一个判断框架）
- **讲解顺序**：
  1. 今天讲一个判断位置的核心框架：估值分位 + 机构持仓 + 趋势位置三维度
  2. 为什么需要这个框架：因为不说位置谈买卖都是空中楼阁
  3. 第一维度：估值分位——看历史百分位，不是看绝对值
  4. 第二维度：机构持仓变化——增仓还是减仓，不是看有没有
  5. 第三维度：趋势位置——上升通道还是下降通道，还是横盘整理
  6. 一句总结：位置不对，努力白费
- **核心讲点**：三维度缺一不可，单一维度判断容易出错

---

## 案例 2

- **适合主题**：怎么判断市场整体情绪热度
- **适合段落**：第 4 段（框架教学）
- **讲解顺序**：
  1. 今天讲一个判断市场情绪的框架：成交量 + 涨跌比 + 涨停家数三指标
  2. 为什么需要这个框架：因为熊市末期和牛市末期情绪特征完全相反
  3. 第一指标：成交量——放量上涨是健康，缩量上涨是虚火
  4. 第二指标：涨跌比——涨跌家数比看市场广度，不是只看指数
  5. 第三指标：涨停家数——连板数量代表短线情绪温度
  6. 一句总结：情绪是皮，逻辑是骨，皮不能代替骨
- **核心讲点**：三指标共振才更有参考价值

## 案例 3

- **适合主题**：基本面选股的四步过滤法
- **适合段落**：第 4 段（框架教学）
- **讲解顺序**：
  1. 今天讲一个基本面选股的四步过滤框架
  2. 为什么需要这个框架：因为市场上股票太多，不过滤看不完
  3. 第一步过滤：行业赛道——选对行业是选对股票的前提
  4. 第二步过滤：财务健康——营收增长、现金流、正利润三选一都要正
  5. 第三步过滤：竞争壁垒——有没有护城河，是品牌还是技术还是规模
  6. 第四步过滤：估值合理——再好的东西买贵了也亏钱
  7. 一句总结：四步过滤过完，剩下的就是值得研究的对象
- **核心讲点**：过滤框架不是为了找出大牛股，是为了减少无效研究

- **核心讲点**：三个维度不是孤立看，要相互印证

# 三、心态认知型

> 结构公式：大家为什么总这样 → 表面原因不是根源 → 真正问题在哪 → 更稳的处理方式 → 一句提醒
> 适合老师风格：陪伴答疑型 / 老师带教型

## 案例 1

- **适合主题**：为什么一买就跌一卖就涨
- **适合段落**：第 2 段（抛核心问题）或第 6 段（第二层认知）
- **讲解顺序**：
  1. 大家为什么总这样：买了就跌，卖了就涨，好像庄家就差你那手
  2. 表面原因不是根源：不是因为你运气差，是因为你被情绪带着走
  3. 真正问题在哪：涨的时候追是因为怕错过，跌的时候卖是因为怕失去
  4. 更稳的处理方式：先定规则再下单，不让价格波动决定你的行为
  5. 一句提醒：你不是在买股票，你是在管理自己的情绪
- **核心讲点**：一买就跌不是庄家针对你，是情绪反射在起作用

---

## 案例 2

- **适合主题**：为什么会卖早了、赚不住钱
- **适合段落**：第 2 段（抛核心问题）
- **讲解顺序**：
  1. 大家为什么总这样：10块钱涨到15块就跑了，结果涨到30块
  2. 表面原因不是根源：不是因为你胆小，是因为你没见过涨
  3. 真正问题在哪：没有给股票足够的上涨空间就急着落袋
  4. 更稳的处理方式：先想清楚这个位置是不是高估，不是想账户盈亏
  5. 一句提醒：卖早是穷思维，卖晚才是富思维
- **核心讲点**：卖出理由应该是高估，不是赚钱

## 案例 3

  3. 真正问题在哪：加仓是在降低均价，但也是在加大赌注
  4. 更稳的处理方式：先判断趋势有没有改变，没改变就持有，改变就止损
  5. 一句提醒：逆势加仓不是摊薄成本，是加大风险暴露
- **核心讲点**：成本价是心理锚点，不是决策依据

# 四、抽象案例型

> 结构公式：给一个场景 → 大家通常怎么理解 → 这种理解哪里会错 → 正确看法是什么 → 回扣前面框架
> 适合老师风格：理性拆解型 / 陪伴答疑型

## 案例 1

- **适合主题**：用「菜市场定价逻辑」讲股票估值
- **适合段落**：第 5 段（案例讲解）
- **讲解顺序**：
  1. 给一个场景：菜市场里黄瓜今天2块，明天2块5，后天3块
  2. 大家通常怎么理解：涨了就是好价格，继续买
  3. 这种理解哪里会错：没考虑供需关系有没有变化，可能只是商贩在炒
  4. 正确看法是什么：要看这两天有没有人大量收购，还是就是有人在控盘抬价
  5. 回扣前面框架：估值和买菜一样要看供需，不能只看价格走势
- **核心讲点**：用生活经验解构金融逻辑

  4. 正确看法是什么：等电梯停稳、开始往上走了再进

# 五、风险提醒型

> 结构公式：先别急 → 这种时候最容易看反 → 为什么会这样 → 正确顺序是什么 → 最后压边界
> 适合老师风格：强观点提醒型 / 理性拆解型

## 案例 1

- **适合主题**：大盘暴跌时该不该抄底
- **适合段落**：第 6 段（风险提醒）
- **讲解顺序**：
  1. 先别急：看到大盘暴跌就想去抄底的人，先看完这段
  2. 这种时候最容易看反：以为跌多了就是机会，结果抄在半山腰
  3. 为什么会这样：因为暴跌往往有惯性，不知道底部在哪
  4. 正确顺序是什么：先看下跌原因，再看是否有支撑位，不急于进场
  5. 最后压边界：暴跌后不抄底，等缩量止跌再做右侧
- **核心讲点**：左侧抄底是高手游戏，普通人等右侧更稳
